# Linux常用命令

## Linux命令格式
```命令 [选项] [参数]```

注意：
    1. 个别命令不符合此格式；
    2. 当有多个“选项”时可以写在一起，比如：-a 和 -l 合写为 -al；
    3. “[]”中括号代表可选；
    4. “简化选项”与“完整选项”，比如：-a 和 --all ，作用是相同的。并不是所有命令都同时支持这两选项，比如：ls 命令只支持 -a；

>超级管理员登录 和 普通用户登录 符号显示区别

超级管理员登录              |    普通用户登录
--------------              |    -----------------
[root@主机名 ~]#            |    [普通用户名@主机名 ~]$
[root@主机名 /]#            |    [普通用户名@主机名 /]$
[root@主机名 当前目录]#      |    [普通用户名@主机名 当前目录]$

**格式：[用户名@主机名 当前目录]超级管理员或普通用户区别标记**

```
注：“~”表示当前登录用户所在目录：
    1. 超级管理员登录，则表示在根目录下的root目录下（/root）；
    2. 普通用户登录，则表示在根目录下的home目录下的普通用户名目
       录下（/home/普通用户名）。
    “#”是root用户特有标记，“$”是普通用户的标记。
```

## cd命令
>cd (change directory)：改变目录
>形式：cd 目录参数
>>目录参数：绝对路径  或 相对路径  或 特殊符号
> * 绝对路径：在Linux中是以 “ **/** ” 开头的路径，一级一级进行递归查找，表示为“ **/二级目录/三级目录/......** ”。比如：cd /root。在任何目录下都可以进入指定位置。不容易出错，推荐使用。
> * 相对路径：以当前所在目录为参考，进行查找，表示为“ **./下级目录/......** ”，“./可以省略”。比如：[root@SL /]cd home 。当前目录一旦改变，原来命令可能无法正确执行。
> * 特殊符号，见下表  



目录参数    |       作用      |       备 注
------------|-------------    |   -------------
/           |       根目录    |Linux系统没有CDEF盘
.           |   当前目录      |   
..          |   上级目录      |   ../.. 上两级目录
~           |   当前用户目录  |①/root ②/home/当前普通用户名
\-          |上一次操作目录   |区别：上一次操作目录、上一级目录

## ls命令
>ls(list)：列出目录下的一般文件或目录
>使用：ls [-参数] [文件或目录]

ls[-参数][文件或目录]|       含义
--------------------|-------------------
ls -a 或 -all       |列出目录下所有文件，包括隐藏文件、特殊文件等，以及 . 和 .. 。
ls -l（等于ll命令）  | 使用详细格式列表         
ls -al              |列出所有文件的详细格式列表
ls -t               | 使用文件和目录的更改时间排序
ls -r               |反向排序
ls -help            |在线帮助
ls -l /home         |列出/home目录下一般文件的详细格式列表
